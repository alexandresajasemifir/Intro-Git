Ce projet est une présentation rapide des fonctionnalité et des bonnes pratiques sur Git.

## Fonctionnalités

- Création de branches

- Création de tags

## Bonnes pratiques

- Utiliser des branches pour chaque fonctionnalité

- Utiliser des tags pour chaque version
